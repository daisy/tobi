Tobi 0.2.3.0 (Beta)

 

Prerequisites

  • DirectX
  • Windows Installer 3.1
  • .NET Framework 3.5 SP1
Download and run setup.exe to install the prerequisites (upon completion, this command should also automatically launch the application). Note: expert users may choose to directly download the .NET Framework and DirectX installers.

Installation and Updates

Once the above prerequisites are installed, you may launch Tobi by clicking on this link, or by using the installed shortcut in the Windows "Start Menu" (in the "DAISY Consortium" sub-menu). The availability of the most recent release of Tobi will be checked each time the application starts, and the most up-to-date version will be downloaded automatically.

Special Instructions

Windows XP and Vista users are advised to install the optional "Windows Automation API" pack, in order to fix performance issues in versions of the .NET framework anterior to version 4.0 (i.e. .NET 3.5 SP1 and before). This is not necessary on Windows 7 or when running .NET 4.0. More information here.
